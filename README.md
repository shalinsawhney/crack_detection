# Tata Steel Crack Detection using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) to detect and classify surface cracks in steel products. The model automates defect inspection and helps improve quality assurance in industrial manufacturing environments.

## Features
- Surface crack detection
- Defect classification
- Deep learning-based image analysis
- Streamlit web interface

## Tech Stack
- Python
- TensorFlow/Keras
- OpenCV
- Streamlit
- NumPy

## Files
- Tata steel Crack Prediction.ipynb
- streamlit_app.py

## Future Improvements
- Real-time camera integration
- Multiple defect category support
